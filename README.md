# GitHub Activity CLI

Check GitHub user activity from your terminal!

## 🚀 Quick Start

```bash
node index.js octocat
```

## 📋 Usage

```bash
node index.js <username>
```

**Examples:**
```bash
node index.js torvalds
node index.js kamranahmedse
```

## ✨ Features
- Shows recent pushes, issues, stars, forks
- Formatted dates
- Error handling
- No dependencies (Node.js 18+ only)

## 📁 Files
- `index.js` - Main CLI program
- `package.json` - Project info
- `README.md` - This file

## Built With
- Node.js 18+ (fetch API)
- GitHub REST API
